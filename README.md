# 🐎 نشر اسب | Asb Publishing
**سر در پی افق‌ها**

اینجا مخزن کدهای «نشر اسب» است؛ پلتفرم اختصاصی من برای انتشار رمان، داستان کوتاه، داستان برق‌آسا و جستار. 

من این وب‌سایت را بدون استفاده از سیستم‌های مدیریت محتوای سنگین (مثل وردپرس) و بر پایه‌ی یک معماری سبک و استاتیک توسعه داده‌ام تا تمرکز اصلی فقط روی خوانایی، سرعت و سادگی باشد.

### 🛠 تکنولوژی‌ها و ابزارهای استفاده‌شده
* **تولیدکننده‌ی سایت (SSG):** اسکریپت اختصاصی نوشته‌شده با پایتون (Python)
* **مدیریت محتوا:** فایل‌های مارک‌دان (`Markdown`) و `Frontmatter` (بدون نیاز به دیتابیس)
* **رندر قالب‌ها:** موتور `Jinja2`
* **فرانت‌اند:** HTML5 معنایی، CSS3 مدرن (بدون فریم‌ورک) و جاوااسکریپت خالص (Vanilla JS)
* **هاست و استقرار:** متصل به **Vercel** برای آپدیت خودکار پس از هر کامیت

### ✨ امکانات اصلی سایت
* تولید و آپدیت خودکار صفحه‌ی اصلی و آرشیوها صرفاً با اضافه کردن یک فایل متنی (`.md`).
* سیستم هشتگ‌گذاری هوشمند با تشخیص و تفکیک خودکار کلمات فارسی و انگلیسی.
* دسته‌بندی و ظاهر شدن خودکار داستان‌ها زیر آیکون‌های اختصاصی ژانر خودشان.
* هدر هوشمند (Smart Header) که هنگام خواندن متن محو می‌شود تا فضای مطالعه بیشتر شود.
* رنگ‌بندی و تایپوگرافی کاملاً استاندارد جهت جلوگیری از خستگی چشم (رعایت اصول Accessibility).
